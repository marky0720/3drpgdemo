# 3drpgdemo
3d rpg demo based irrlicht
2010 年的作品 
irrlicht 引擎的使用比较多，模型加载动作，拾取，添加删除，例子特效，水模拟，瀑布模拟. gui
适合irrlicht入门学习。

编译
依赖irrlicht1.6  raknet 对应源码已经打包提交。下载后自行设置安装。
